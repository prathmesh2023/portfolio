Project Overview

The vdfengineering.co.in is the official website of V.D.F School of Engineering and Technology and regularly maintained by me . I have created this website as per the requirements and frequently updating it

This website having features of dynamic content managed by admin with admin dashboard and we can upload or update pages, news, media etc without coding and realtime unique visitors of website can be tracked .

For developing front-end of this project i used html , css for defining structure and styling pages, Bootstrap used for making website responsive .
For back-end Django (python) framework is used with some packages of python and any database can be flexible with that project with the help of database client library .

Currently this website is hosten on AWS(EC2) on Ubuntu instance and apache2 server is used to host the project. This project is completely designed and developed from scratch .
